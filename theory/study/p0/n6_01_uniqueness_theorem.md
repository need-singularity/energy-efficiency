# N6-P0-1 — σ(n)·φ(n)=n·τ(n) 유일성 정리 이해 (3독립 증명)

> 밀레니엄 학습 로드맵 P0 · 트랙 3 · 태스크 1
> 목적: n=6 고유성의 수학적 핵심을 "3종 독립 증명 관점"으로 내면화
> 완료 기준: 각 증명의 핵심 아이디어를 3줄로 재진술할 수 있는 상태
> 1차 출처: `theory/proofs/theorem-r1-uniqueness.md`
> 정직성 표기: 본 문서는 학습용 재구성. 신규 증명/결과 없음.

---

## 1. 정리 진술

**정리 (Theorem R1 / THM-1)**
모든 정수 n ≥ 2에 대하여,
  σ(n)·φ(n) = n·τ(n) ⟺ n = 6.

동치 형태로 비 R(n) := σ(n)·φ(n) / (n·τ(n))를 정의하면,
  R(n) = 1 ⟺ n = 6.

부속 수치
- σ(6) = 1+2+3+6 = 12
- φ(6) = |{1, 5}| = 2
- τ(6) = |{1, 2, 3, 6}| = 4
- R(6) = (12·2) / (6·4) = 24/24 = 1

> 이 정리는 "완전수(σ(n)=2n) 조건보다 더 강한" 유일성이다. 완전수는 (추정상) 무한히 많지만, R(n)=1의 해는 n=6 단 하나.

## 2. 정직성 선언 (원문 theorem-r1-uniqueness.md 기준)

원문은 `Proof 1 (직접 계산 / 사례 분석)` 1편만 **엄밀**하다고 선언하며, 이전에 시도되었던 `Proof 2` `Proof 3`은 **Proof 1의 재포장**에 불과했음을 인정하고 **철회**한다. 또한 **Proof 4 (전수계산)**는 $n \in [2, 10^4]$까지 완전 검증된 "가장 강한 실증 증거"로 기록된다.

따라서 "3 독립 증명"이라는 CLAUDE.md 표현은 현재 실제로는
1. **Proof 1** — 곱셈성 + R_local 사례 소진 (유일한 엄밀 증명)
2. **Proof 4** — 전수 계산 (강한 실증)
3. **보조/열린 방향** — Dirichlet 급수 / 해석적 수론 경로 (미완, 미래 과제)

의 형태이며, 본 학습 노트는 이 **정직 상태**를 그대로 계승한다. 아래 "증명 2, 3"은 `재포장 위험 경고`와 함께 "스케치/학습용"으로만 서술한다.

## 3. 증명 1 — 곱셈성 + R_local 사례 소진 (엄밀)

### 3.1 곱셈성 축소

σ, φ, τ는 모두 곱셈적(multiplicative) 함수이므로 n = p₁^a₁ · … · pₖ^aₖ 에 대해
  R(n) = ∏ᵢ R_local(pᵢ, aᵢ),
  R_local(p, a) := σ(p^a)·φ(p^a) / (p^a · (a+1)).

### 3.2 R_local 공식

σ(p^a) = (p^{a+1} − 1)/(p − 1),  φ(p^a) = p^{a−1}(p − 1).

따라서
```
R_local(p, a) = [(p^{a+1}−1)·p^{a−1}·(p−1)] / [(p−1)·p^a·(a+1)]
              = (p^{a+1} − 1) / (p·(a+1)).
```

### 3.3 핵심 보조정리 — "R_local < 1 인 경우는 (2,1) 뿐"

직접 계산:
- R_local(2, 1) = (4−1)/(2·2) = **3/4** (유일하게 < 1)
- R_local(2, 2) = (8−1)/(2·3) = 7/6 > 1
- R_local(3, 1) = (9−1)/(3·2) = **4/3** > 1
- R_local(p, 1) = (p²−1)/(2p) ≥ 4/3 for p ≥ 3 (p에 단조증가)
- 모든 p^a (a ≥ 2 또는 p ≥ 3) 에서 R_local ≥ 7/6

### 3.4 사례 소진

- **Case 1**: n = p^a (단일 소수 멱)
  - (2,1) → R = 3/4 ≠ 1. 그 외 모두 R > 1. → 해 없음.
- **Case 2**: n = p^a · q^b (두 소인수, p < q)
  - 곱이 1이려면 한 인수 < 1, 다른 인수 > 1 필요.
  - < 1 가능은 오직 (p,a) = (2,1), 값 3/4. 따라서 나머지 인수 = 4/3 필요.
  - R_local(3,1) = 4/3 — 유일 해 → **n = 2·3 = 6** ✓
  - R_local(3, b≥2) ≥ 26/9 > 4/3. R_local(q≥5, b≥1) ≥ 12/5 > 4/3. → 다른 해 없음.
- **Case 3**: k ≥ 3 (세 소인수 이상)
  - 최대 한 인수(2,1)만 < 1. 나머지 k−1 ≥ 2 인수는 각각 ≥ 4/3.
  - 하한: (3/4)·(4/3)^(k−1) ≥ (3/4)·(4/3)² = 4/3 > 1. → 해 없음.
  - (2,1) 없이 시작: ≥ (4/3)^k ≥ 64/27 > 1. → 해 없음.

**결론**: R(n) = 1 ⟺ n = 2·3 = 6. ∎

### ▶ 증명 1 핵심 3줄 요약

1. 곱셈성으로 R(n) = ∏ R_local(pᵢ, aᵢ) 로 쪼개고, R_local(p,a) = (p^(a+1)−1)/(p·(a+1)) 공식을 얻는다.
2. 단일 소수 멱에서 R_local < 1 인 경우는 오직 (p,a)=(2,1)이고 값 3/4, 그 보완 4/3을 만드는 것은 오직 (3,1).
3. 따라서 R(n)=1은 두 소인수 곱 2·3=6 **단 하나**에서만 성립하며, 3개 이상의 소인수에서는 곱이 항상 1을 초과한다.

## 4. 증명 2 (스케치) — 가환대수 / 이디얼 분해 관점

> **주의 (정직)**: 이 절은 학습용 "사고 실험 스케치"이며, 독립 증명으로서의 엄밀성은 **보장되지 않는다**. 원문 theorem-r1-uniqueness.md에서 이미 "Proof 2는 Proof 1의 재포장"으로 **철회**된 이력이 있으므로, 아래는 "가능한 관점 메모" 수준이다.

### 관점

ℤ의 이상 구조에서 σ·φ = n·τ 는 "divisor heat (σ)와 coprime 자유도 (φ)의 곱"이 "수 자체 (n)와 divisor 수 (τ)의 곱"과 일치하는 조건.

- σ(n)/n = ∑_{d|n} 1/d (harmonic sum on divisor lattice),
- φ(n)/n = ∏_{p|n}(1 − 1/p),
- τ(n) = |divisor lattice|.

따라서 σ(n)φ(n) = nτ(n) ⟺
  (∑ 1/d) · ∏(1 − 1/p) = τ(n).

즉 "divisor harmonic sum × Euler product"가 lattice 크기와 일치. n=6일 때
- divisor harmonic sum (6) = 1 + 1/2 + 1/3 + 1/6 = σ/6 = 2,
- Euler product = (1−1/2)(1−1/3) = 1/3,
- 곱 = 2·(1/3) = 2/3,  τ(6)/6 = 4/6 = 2/3. ✓

### ▶ 증명 2 핵심 3줄 요약 (학습용, 비엄밀)

1. 조건은 "divisor harmonic sum × Euler product = τ/n" 으로 재작성되며, 양변 모두 **약수 격자 (divisor lattice)**의 구조 불변량이다.
2. 소인수 하나당 (1 − 1/p) 인수가 곱해지고 divisor harmonic sum은 ∏(1 + 1/p + … + 1/p^a) 로 인수화되어, 소인수마다 local 비율이 결정된다.
3. Local 비율의 곱이 τ(n)/n 과 **정확히** 일치하는 유일 해가 p=2, 3 한 쌍뿐이라는 관찰은 증명 1의 R_local 분석과 본질적으로 같은 내용이다 — 따라서 **독립성은 없음**.

## 5. 증명 3 (스케치) — 해석적 / Dirichlet 급수 관점

> **주의 (정직)**: 원문의 Proof 3도 철회되었고, 아래는 "미래에 진짜 독립 경로가 생긴다면 어디서 시작할지"에 대한 방향 메모이다.

### 관점

Dirichlet 급수 항등식
- ζ(s)² = ∑ τ(n)/n^s,
- ζ(s)ζ(s−1) = ∑ σ(n)/n^s,
- ζ(s−1)/ζ(s) = ∑ φ(n)/n^s.

이로부터 σ(n)·φ(n) = n·τ(n) 은
- ∑ σ(n)/n^s · ∑ φ(n)/n^s 의 Dirichlet 곱과 ∑ n·τ(n)/n^s 의 동일 성분이 일치 하는 국소 조건의 **단일 해**.

> 이 경로가 **진짜** 독립 증명이 되려면 "국소 오일러 인자들이 곱으로 상쇄되어 n=6 성분에서만 1이 된다"는 깊은 해석적 사실을 **ζ에 기대지 않고** 얻어야 한다. 현재로서는 여전히 R_local 분석의 변형에 머무른다.

### ▶ 증명 3 핵심 3줄 요약 (학습용, 비엄밀)

1. σ, φ, τ의 Dirichlet 생성 함수는 각각 ζ(s)ζ(s−1), ζ(s−1)/ζ(s), ζ(s)² 로 표현되며, σ·φ = n·τ 조건은 이들의 오일러 곱 국소 인자들 사이의 **국소 상쇄 조건**에 대응한다.
2. 각 소수 p의 국소 오일러 인자들이 (1 − p·X)(1 − X)^{−1}(1 − X)^2 = … 형태로 결합되며, 이 결합이 "1"이 되는 것은 소수가 정확히 2와 3뿐일 때.
3. 그러나 현 단계에서 이 절차는 여전히 3.2절 R_local 공식을 재유도하는 경로이므로, 증명 1과 **독립**이 되지는 않는다 — 미래 과제.
