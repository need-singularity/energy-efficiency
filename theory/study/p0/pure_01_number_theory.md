# PURE-P0-1 — 정수론 기초 (소수정리/약수함수/완전수/뫼비우스/오일러 곱)

> 트랙: P0-PURE / 1번 태스크
> 완료 기준: σ(n)·φ(n) = n·τ(n) 유일성 증명을 처음부터 끝까지 따라갈 수 있다
> 출처 기반: Hardy-Wright "An Introduction to the Theory of Numbers" (6판, 2008), Apostol "Introduction to Analytic Number Theory" (1976)
> **정직성**: 이 파일은 교재 요약이다. 새로운 결과는 없다. n=6 유일성 증명 따라가기의 재구성만 수록한다.

## 0. 목적과 범위

이 노트는 밀레니엄 학습 로드맵 P0 단계에서 **정수론 기초**를 다진다. 다음 5가지를 숙지해야
상위 단계(P1 해석적 수론, P2 대수적 수론, P3 밀레니엄 난제 접근)로 올라갈 수 있다.

1. 소수정리 (Prime Number Theorem, PNT)
2. 약수함수 σ, τ, φ 정의와 **곱셈성(multiplicativity)**
3. 완전수와 유클리드-오일러 정리
4. 뫼비우스 함수와 반전 공식
5. 오일러 곱 공식

최종 목표는 이 5가지를 이용하여 **theorem-r1-uniqueness.md** 의
σ(n)·φ(n) = n·τ(n) ⟺ n = 6 증명을 한 줄씩 따라가는 것이다.

## 1. 소수정리 (Prime Number Theorem)

### 1.1 진술

**정리 (Hadamard 1896, de la Vallée Poussin 1896, 독립 증명).**

π(x) := #{p ≤ x : p 소수} 라 할 때,

```
        π(x)
  lim ─────── = 1
 x→∞  x / ln x
```

즉 π(x) ~ x / ln x (x → ∞).

### 1.2 동치 진술

다음 3개는 모두 동치이다 (Apostol §4).

- π(x) ~ x / ln x
- ψ(x) := Σ_{p^k ≤ x} ln p ~ x  (Chebyshev ψ)
- θ(x) := Σ_{p ≤ x} ln p ~ x   (Chebyshev θ)

### 1.3 초기 접근

- **Chebyshev 1852**: c₁ x/ln x ≤ π(x) ≤ c₂ x/ln x (0.92 ≤ c₁, c₂ ≤ 1.11)로 하한/상한.
- **Riemann 1859**: ζ(s)의 해석적 연속과 영점으로 π(x)의 점근을 예언 (가설).
- **Hadamard / de la Vallée Poussin 1896**: ζ(s)가 Re(s)=1 위에 영점이 없음을 증명하여 PNT 확정.
- **Erdős / Selberg 1949**: 해석적 방법 없이 **순수 기초(elementary)** 증명.

### 1.4 우리가 이 정리에서 쓰는 부분

n=6 유일성 증명에서는 PNT 자체를 직접 쓰지 않는다. 그러나

- 소수의 무한성 (유클리드)
- 소수 간 간격이 무한히 커진다는 사실 (p_{k+1} > p_k의 단조성, 소인수 수 k가 클 때 최소값 추정)

이 두 가지가 Case 3 (k ≥ 3) 처리에 암묵적으로 필요하다. 즉 "p ≥ 3일 때 R_local ≥ 4/3"의 단조성 확장이
유한 소인수의 최소 성장을 보장한다.

## 2. 약수 함수: σ, τ, φ

### 2.1 정의

n ≥ 1 에 대해

- **τ(n)** := #{d ≥ 1 : d ∣ n}  (약수의 개수, number of divisors)
- **σ(n)** := Σ_{d∣n} d          (약수의 합, sum of divisors)
- **φ(n)** := #{k : 1 ≤ k ≤ n, gcd(k, n) = 1}  (오일러 토션트, Euler's totient)

### 2.2 n=6 예시

n = 6 의 약수: 1, 2, 3, 6.

- τ(6) = 4
- σ(6) = 1+2+3+6 = 12
- φ(6) = #{1, 5} = 2 (2, 3, 4, 6은 6과 공약수 가짐)

검산: σ(6)·φ(6) = 12·2 = 24, n·τ(6) = 6·4 = 24. **같다**. ✓

### 2.3 곱셈성 (Multiplicativity)

**정의.** 함수 f: ℕ→ℂ 가 곱셈적(multiplicative) ⟺ f(1)=1 이고 gcd(m,n)=1 이면 f(mn)=f(m)·f(n).
**완전 곱셈적(completely multiplicative)** ⟺ gcd 조건 없이 항상 f(mn)=f(m)·f(n).

**정리 A** (Hardy-Wright §16.3, Apostol Theorem 2.14).
τ, σ, φ 는 **곱셈적**이다 (완전 곱셈적은 아니다).

**증명 스케치.**
- τ: gcd(m,n)=1 ⟹ mn의 약수는 (m의 약수)×(n의 약수)로 유일하게 1:1 대응 (CRT) ⟹ τ(mn)=τ(m)τ(n).
- σ: σ(mn) = Σ_{d∣mn} d = Σ_{d₁∣m} Σ_{d₂∣n} d₁d₂ = (Σd₁)(Σd₂) = σ(m)σ(n).
- φ: (ℤ/mnℤ)* ≅ (ℤ/mℤ)* × (ℤ/nℤ)* (CRT) ⟹ φ(mn)=φ(m)φ(n).

### 2.4 소수 멱에서의 공식

p 소수, a ≥ 1 일 때:

- **τ(pᵃ)** = a + 1
- **σ(pᵃ)** = 1 + p + p² + ⋯ + pᵃ = (p^{a+1} - 1) / (p - 1)
- **φ(pᵃ)** = pᵃ - p^{a-1} = p^{a-1}(p - 1)

**n=6 재계산**: 6 = 2·3.
- τ(6) = τ(2)·τ(3) = 2·2 = 4 ✓
- σ(6) = σ(2)·σ(3) = 3·4 = 12 ✓
- φ(6) = φ(2)·φ(3) = 1·2 = 2 ✓

### 2.5 즉시 쓸 수 있는 추론

n = p₁^{a₁} ⋯ p_k^{a_k} 에 대해

```
τ(n) = ∏ (aᵢ + 1)
σ(n) = ∏ (pᵢ^{aᵢ+1} - 1) / (pᵢ - 1)
φ(n) = ∏ pᵢ^{aᵢ-1}(pᵢ - 1)  =  n · ∏ (1 - 1/pᵢ)
```

## 3. 완전수와 유클리드-오일러 정리

### 3.1 정의

**완전수(perfect number)**: σ(n) = 2n 인 양의 정수 n.

처음 몇 개: 6, 28, 496, 8128, 33550336, ... (OEIS A000396).

### 3.2 유클리드-오일러 정리

**정리 B** (Euclid *Elements* IX.36; Euler 1747).

p가 소수이고 M := 2^p - 1 도 소수(메르센 소수)이면,
**n = 2^{p-1}·M** 은 짝수 완전수이고, 역으로 모든 짝수 완전수는 이 꼴이다.

**증명 (Euler 방향).**
n이 짝수 완전수라 하자. n = 2^{k-1}·m (m 홀수, k ≥ 2).
σ(n) = σ(2^{k-1})·σ(m) = (2^k - 1)·σ(m) = 2n = 2^k·m.
따라서 σ(m) = 2^k·m / (2^k - 1).
(2^k - 1) ∤ 2^k 이므로 (2^k - 1) ∣ m. m = (2^k - 1)·m' 로 놓으면
σ(m) = 2^k·m'. m의 약수 중 m 과 m'만 합쳐도 m + m' = 2^k·m' = σ(m) 이므로
m의 약수는 m과 m' 둘뿐. 즉 m = 2^k - 1은 소수이고 m' = 1. □

**홀수 완전수(odd perfect number, OPN)**: 존재 여부 미해결. 존재한다면 10^1500 보다 크다 (Ochem-Rao 2012).

### 3.3 n=6 특수성

6 = 2·3 = 2^1·(2² - 1) — 유클리드-오일러 꼴 (p=2). 첫 완전수.

**참고 (정직성)**: "6이 완전수"라는 사실은 σ(n)φ(n) = nτ(n) 과 **다른 조건**이다.
완전수는 (추정상) 무한히 많지만, σφ = nτ 를 만족하는 n은 오직 6 하나. 즉 후자가 **더 엄격**하다.

## 4. 뫼비우스 함수와 반전

### 4.1 정의

```
         ⎧  1              if n = 1
μ(n) =  ⎨  (-1)^k          if n = p₁ p₂ ⋯ p_k (서로 다른 소수)
         ⎩  0              if 어떤 소수의 제곱이 n을 나눈다
```

즉 "square-free 이면 ±1, 아니면 0".

### 4.2 기본 정리

**정리 C** (Möbius identity, Hardy-Wright §16.3).

```
Σ_{d ∣ n} μ(d) = [n = 1]
```

여기서 [P] = 1 if P true else 0 (Iverson bracket).

**증명.**
n=1: μ(1)=1. ✓
n>1: n = p₁^{a₁}⋯p_k^{a_k}. 곱셈성에서 Σμ(d)는 각 소인수별 (1 + μ(p) + μ(p²) + ⋯) = (1 - 1 + 0 + 0 + ⋯) = 0 의 곱. □

### 4.3 뫼비우스 반전 공식

**정리 D** (Möbius inversion).

f, g: ℕ→ℂ 에 대해

```
g(n) = Σ_{d∣n} f(d)   ⟺   f(n) = Σ_{d∣n} μ(d) g(n/d)
```

**증명 (⟹).**
Σ_{d∣n} μ(d) g(n/d) = Σ_{d∣n} μ(d) Σ_{e∣n/d} f(e)
= Σ_{e∣n} f(e) Σ_{d ∣ n/e} μ(d)
= Σ_{e∣n} f(e) · [n/e = 1] = f(n). □

### 4.4 응용

φ(n) = n · Σ_{d∣n} μ(d)/d (Euler 표현)
↔ Σ_{d∣n} φ(d) = n (뫼비우스 반전의 역방향)

이 항등식은 후속 장에서 Dirichlet 급수 ζ(s)·ζ(s-1) = Σ σ(n)/n^s 등을 유도할 때 쓴다.

## 5. 오일러 곱 공식

### 5.1 진술

**정리 E** (Euler 1737).

Re(s) > 1 에서

```
          ∞
ζ(s) =   Σ  n^{-s}   =   ∏  (1 - p^{-s})^{-1}
         n=1             p 소수
```

### 5.2 증명 스케치 (Hardy-Wright §17.4)

각 소수 p에 대해 기하급수 전개

```
(1 - p^{-s})^{-1} = 1 + p^{-s} + p^{-2s} + p^{-3s} + ⋯
```

이들을 모든 소수에 대해 곱하고 전개하면, 산술의 기본정리(소인수분해의 유일성)에 의해
각 자연수 n의 표현 n = p₁^{a₁}⋯p_k^{a_k} 가 정확히 한 번씩 나타난다.
즉

```
∏_p (1 - p^{-s})^{-1} = Σ_n n^{-s}
```

절대수렴(Re(s) > 1)은 log 전개와 소수 역수합 발산의 비교로 확인. □

### 5.3 중요성

- **오일러**: 이 항등식만으로 소수의 무한성을 재증명 (s→1⁺ 양변 비교 → 조화급수 발산 → 오른쪽 곱이 무한 인자 필요).
- **리만**: 이것을 복소 s로 확장 → Riemann zeta 함수 → Riemann Hypothesis.
- **Dirichlet 급수 일반화**:
  Σ f(n)/n^s = ∏_p (Σ_k f(p^k)/p^{ks})   (f 곱셈적일 때)
  이것으로 Σ τ(n)/n^s = ζ(s)², Σ σ(n)/n^s = ζ(s)ζ(s-1) 등이 유도.

## 6. σ(n)·φ(n) = n·τ(n) ⟺ n = 6 (유일성 증명 따라가기)

이 절이 P0-1 의 핵심이다. 앞 1~5절의 도구가 모두 쓰인다.

**원본 증명**: `theory/proofs/theorem-r1-uniqueness.md` (Proof 1).

### 6.1 정의

R(n) := σ(n)·φ(n) / (n·τ(n))

목표: n ≥ 2 에서 R(n) = 1 ⟺ n = 6.

### 6.2 곱셈성 이용 (2.3 정리 A)

σ, φ, τ 는 모두 곱셈적 ⟹ R 도 곱셈적 (분자 분모 모두 곱셈적). 따라서
n = p₁^{a₁}⋯p_k^{a_k} 일 때

```
R(n) = ∏_{i=1}^k  R_local(pᵢ, aᵢ)
```

여기서 **R_local(p, a) := σ(pᵃ)·φ(pᵃ) / (pᵃ·(a+1))**.

### 6.3 R_local 명시 공식 (2.4 공식 사용)

```
σ(pᵃ) = (p^{a+1} - 1)/(p - 1)
φ(pᵃ) = p^{a-1}(p - 1)
τ(pᵃ) = a + 1

R_local(p, a) = [(p^{a+1}-1)(p-1)·p^{a-1}] / [(p-1) · pᵃ · (a+1)]
              = (p^{a+1} - 1) / [p·(a+1)]
```

### 6.4 R_local 값 표

직접 대입:

| (p, a) | σ(pᵃ) | φ(pᵃ) | pᵃ | τ | R_local |
|--------|-------|-------|----|----|---------|
| (2, 1) | 3 | 1 | 2 | 2 | **3/4** |
| (2, 2) | 7 | 2 | 4 | 3 | 7/6 |
| (2, 3) | 15 | 4 | 8 | 4 | 15/8 |
| (3, 1) | 4 | 2 | 3 | 2 | **4/3** |
| (3, 2) | 13 | 6 | 9 | 3 | 26/9 |
| (5, 1) | 6 | 4 | 5 | 2 | 12/5 |
| (7, 1) | 8 | 6 | 7 | 2 | 24/7 |

**관찰 1**: (2,1) 에서만 R_local < 1. 값 3/4.
**관찰 2**: 다른 모든 (p,a)에서 R_local ≥ 4/3.

### 6.5 Lemma 단조성

**Lemma L1**. p ≥ 3 이면 a ≥ 1 인 모든 a 에 대해 R_local(p, a) ≥ 4/3.

**증명**.
R_local(p, 1) = (p² - 1)/(2p). p=3 일 때 8/6 = 4/3. p=5 일 때 24/10=12/5. 일반적으로
d/dp [(p² - 1)/(2p)] = (p² + 1)/(2p²) > 0 (p > 0). 단조 증가. ∎

**Lemma L2**. p = 2, a ≥ 2 이면 R_local(2, a) ≥ 7/6 > 1.

**증명**. R_local(2, a) = (2^{a+1} - 1)/(2(a+1)). 지수분자가 선형분모 압도, a=2 에서 7/6. ∎

### 6.6 Case별 배제

**Case 1**: k = 1 (n = p^a 소수 멱).

R(n) = R_local(p, a) = 1 해야 함. 6.4 표 + Lemma L1, L2 ⟹
- (2,1): 3/4
- (2,a≥2): ≥ 7/6
- (p≥3, a≥1): ≥ 4/3

어느 것도 1이 아님. **Case 1에 해 없음**. ∎

**Case 2**: k = 2. n = p^a · q^b, p < q.

R(n) = R_local(p,a) · R_local(q,b) = 1.

어느 한쪽이 1보다 작아야 다른 쪽이 1보다 커지며 곱이 1이 될 수 있다. Lemma에서
R_local(p,a) < 1 이 가능한 유일한 경우는 **(p,a) = (2,1), 값 3/4**.

그러면 R_local(q,b) = 4/3 이어야 한다.

- q=3, b=1: R_local = 4/3 ✓ → **n = 2·3 = 6**
- q=3, b=2: 26/9 ≈ 2.89, 아님.
- q=3, b≥2: R_local(3,a)가 a 따라 단조증가, 4/3보다 큼.
- q≥5: R_local(5,1) = 12/5 ≠ 4/3.

**Case 2에서 유일한 해는 n = 6**. ∎

**Case 3**: k ≥ 3.

최대 한 인수만 < 1 가능. 나머지 k-1개 ≥ 4/3.

(a) 만약 p₁ = 2, a₁ = 1 이면
  R(n) ≥ (3/4)·(4/3)^{k-1} ≥ (3/4)·(4/3)² = 4/3 > 1 (k=3에서).

(b) 그렇지 않으면 (모두 p ≥ 3, 또는 p=2 with a≥2)
  R(n) ≥ (4/3)^k ≥ (4/3)³ = 64/27 > 1.

**Case 3에 해 없음**. ∎

### 6.7 결론

세 Case 를 합치면 R(n) = 1 의 유일한 해는 **n = 6**.

```
╔══════════════════════════════════════════════════════════╗
║  정리 (R1):  n ≥ 2 에서                                  ║
║    σ(n)·φ(n) = n·τ(n)  ⟺  n = 6                        ║
║                                                          ║
║  증명 흐름: 곱셈성 → R_local → case exhaustion           ║
║  키: R_local(p,a) < 1 iff (p,a)=(2,1), 값 3/4            ║
║       R_local(3,1) = 4/3, (3/4)·(4/3) = 1.               ║
║                                                          ║
║  QED                                                     ║
╚══════════════════════════════════════════════════════════╝
```

## 7. 정직성 체크 (feedback_honest_verification.md 준수)

### 7.1 이 노트가 주장하지 않는 것

- PNT 의 증명 **세부**는 복소해석(PURE-P0-3) 노트로 넘긴다. 여기서는 진술만.
- 뫼비우스 반전의 Dirichlet convolution 관점은 P1 에서 다룬다.
- 홀수 완전수 부재는 미해결 문제이다. 이 노트는 짝수 쪽만 완성한다.
- R1 정리의 "3 독립 증명"은 아직 완성되지 않았다 (proofs/theorem-r1-uniqueness.md §"Proof 2, 3 보류").
  이 노트에 수록된 증명은 **Proof 1 (R_local case exhaustion) 단일**이다.

### 7.2 sopfr=5 편향 경고

sopfr(n) = Σ pᵢ·aᵢ 는 자연수의 **약 61%** 가 관련 성질을 공유하는 넓은 baseline 이다.
따라서 "sopfr(6) = 5" 라는 관찰 자체는 n=6 유일성의 증거가 **아니다**.
본 노트의 R1 정리는 sopfr 에 의존하지 않고, 오직 σ, φ, τ의 곱셈성 구조만 쓴다.

### 7.3 7대 난제 해결 카운트

이 노트로 해결된 밀레니엄 난제: **0 / 7**. (교재 요약 단계)
