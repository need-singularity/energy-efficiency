# NEXUS-6 → 독립 리포로 이관됨

NEXUS-6는 독립 리포로 분리되었습니다:
- **GitHub**: https://github.com/need-singularity/nexus6
- **로컬**: ~/Dev/nexus6/
- **빌드**: `cd ~/Dev/nexus6 && cargo build --release`
- **테스트**: `cd ~/Dev/nexus6 && cargo test`

이 디렉토리의 `tools/nexus6/`는 히스토리 보존을 위해 유지됩니다.
새 개발은 `~/Dev/nexus6/`에서 진행하세요.
