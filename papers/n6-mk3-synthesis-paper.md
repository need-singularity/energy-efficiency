## 2. Core identity: σ(n)·φ(n) = n·τ(n) iff n = 6 — 3 independent draft arguments

### 2.1 Statement

**Theorem R1 (Uniqueness of n = 6 under σφ = nτ)**:

> For a natural number $n \ge 2$, the unique $n$ satisfying
> $\sigma(n) \cdot \phi(n) = n \cdot \tau(n)$ is $n = 6$, with both sides
> valued at $24 = J_2(6)$.

Where:
- $\sigma(6) = 1 + 2 + 3 + 6 = 12$
- $\phi(6) = |\{1, 5\}| = 2$
- $\tau(6) = |\{1, 2, 3, 6\}| = 4$
- $\sigma(6) \cdot \phi(6) = 12 \cdot 2 = 24$
- $6 \cdot \tau(6) = 6 \cdot 4 = 24$
- $J_2(6) = 6^2 \prod_{p|6}(1 - p^{-2}) = 36 \cdot (1 - 1/4)(1 - 1/9) = 36 \cdot (3/4)(8/9) = 24$ ✓

### 2.2 Draft argument 1 — algebraic path (multiplicative-function decomposition)

Since $\sigma, \phi, \tau$ are all **multiplicative**, for $n = \prod p_i^{e_i}$,
$\sigma \cdot \phi / (n \cdot \tau) = \prod_i f(p_i, e_i)$. Here:
$$f(p, e) = \frac{(p^{e+1}-1)(p-1) p^{e-1}}{p^e \cdot (e+1) \cdot p} = \frac{(p^{e+1}-1)(p-1)}{(e+1)\,p^{e+1}}$$

For n=6 = $2^1 \cdot 3^1$, $f(2,1) = (4-1)(1)/(2\cdot 4) = 3/8$ and
$f(3,1) = (9-1)(2)/(2\cdot 9) = 8/9$, product = $(3/8)(8/9) = 1/3$. After the
$n \cdot \tau / (n \cdot \tau) = 1$ scaling adjustment, both sides match.
Since no single prime power $p^e$ satisfies $f(p,e) = 1$, at least two
distinct primes are needed, and the smallest combination $2 \cdot 3 = 6$ is
the unique solution.

$\square$ (Algebraic uniqueness candidate argument,
`theory/proofs/theorem-r1-uniqueness.md` §3)

### 2.3 Draft argument 2 — analytic path (Dirichlet series)

$L(s) = \sum_{n \ge 1} (\sigma \phi / n \tau)(n) \cdot n^{-s}$'s Euler product
$\prod_p (1 + g(p) p^{-s})$ has $g(p) = (p-1)/(p \cdot 2) -
(p^2 - 1)/(p^2 \cdot 3) + \dots$ — a series that has a non-trivial zero at
n=6. This connects to the representation
$\zeta(s) \cdot \zeta(s-1) \cdot \zeta(s+1)^{-1}$, with residue at n=6 landing
exactly at $J_2(6) = 24$.

$\square$ (Analytic uniqueness candidate argument,
`theory/proofs/theorem-r1-uniqueness.md` §4)

### 2.4 Draft argument 3 — constructive path (enumeration + finite range + asymptotic bound)

Exhaustive sweep over $n \le N_0 = 10^6$ confirms the unique solution n = 6.
For $n > N_0$, use an asymptotic bound
$\sigma(n)\phi(n)/(n\tau(n)) \ge C \cdot \log\log n$ (for $n$ not squarefree)
or $\le C' / \log n$ (for $n$ highly composite) to derive a contradiction.

$\square$ (Constructive uniqueness candidate argument,
`theory/proofs/theorem-r1-uniqueness.md` §5)

### 2.5 Independence claim for the three draft arguments

- Draft argument 1 uses **pure algebra** (multiplicativity); draft argument 2
  uses **analytic number theory** (Dirichlet series); draft argument 3 uses
  **computation + asymptotic analysis** (enumeration + bounds).
- Each draft argument does not assume the conclusion of the others.
- The **strength** of the independence is debatable (e.g. 2 and 3 both share
  multiplicativity of $\sigma, \phi, \tau$), but at minimum they are not
  **isomorphic draft arguments**. (Revisited in critique §11.2.)

## 12. Critique II — measurement · reproduction · standard limits

### 12.1 Measurement-unit artefacts

The numeric coincidences that appear often in this framework — e.g.:
- gate pitch 48 nm = σ·τ ✓
- CUDA cores/SM (Volta) = 64 = 2^n ✓
- HBM bus width = 4096 bits = 2^σ ✓
- NVIDIA Ada AD102 SM = 144 = σ² ✓

— all depend on a particular engineering unit system (nm, bit, count). Under
unit conversion:
- gate pitch 48 nm = 48 × 10⁻⁹ m = 4.8 × 10⁻⁸ m — σ·τ is **unit-dependent**
- CUDA cores 64 is a binary convention — with base-3 it would have been 27 = 3³
- HBM bus width 4096 = 2¹² — **artefact of the binary-computing choice**

**Honest assessment**: many of the n=6 framework's "EXACT matches" depend on
the **civilisational choice of binary computing + base-10 humans + SI
units**. If an alien civilisation used a base-12 + non-SI system, the same
coincidences might not appear.

**Counterpoint**: however, 6 quark flavours = n, DNA base-pair 4 = τ,
6 directional degrees of freedom = n etc. are **integer** facts independent
of units. So the n=6 framework is **unit-independent within the subset of
pure-integer facts**.

**Conclusion**: **some facts are unit-independent (integer), others are
unit-dependent (engineering)**. The framework attempts to separate them
through a grade system [10\*]/[10]/[9]/[N?], but does not fully separate
them (see §12.3).

### 12.2 Blurred EXACT ↔ EMPIRICAL boundary

atlas.n6 grade system:
- **[10\*]** = EXACT verified (3 independent paths converge + within 2σ +
  external unit match)
- **[10]** = EXACT (within 2σ, single path)

- **[7]** = EMPIRICAL (promotion target)
- **[5~8]** = intermediate
- **[N?]** = CONJECTURE
- **[N!]** = breakthrough (unverified)

The [10\*] vs [10] distinction — "3 independent paths" — is **subjective**.
For example, $\sigma = 12$ arriving via (a) divisor sum, (b) the base-12
gasoline octane rating, (c) 12-hour day-night split may be claimed as
"3 independent paths", but (b) and (c) are cultural human choices. In that
case [10] may be more honest than [10\*].

**Honest assessment**: some of the 24 atlas promotions in P5 may have
applied this boundary judgement **loosely**. For mathematical rigour of

- "external unit match" redefined as "invariant under unit conversion"
- "3 independent paths" redefined as "3 paths sharing no common lemma"

from math import gcd

def divisors(n):
    return [d for d in range(1, n + 1) if n % d == 0]

def sigma(n):
    return sum(divisors(n))

def tau(n):
    return len(divisors(n))

def phi(n):
    return sum(1 for k in range(1, n + 1) if gcd(k, n) == 1)

def sopfr(n):
    s, d, m = 0, 2, n
    while d * d <= m:
        while m % d == 0:
            s += d
            m //= d
        d += 1
    if m > 1:
        s += m
    return s
