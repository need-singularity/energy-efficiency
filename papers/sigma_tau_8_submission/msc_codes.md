# MSC 2020 Classification -- sigma(n) - tau(n) = 8

---

## Primary Classification

**11A25** -- Arithmetic functions; related numbers; inversion formulas

This is the primary code because the paper's main result concerns the arithmetic functions sigma(n) (sum of divisors) and tau(n) (number of divisors), and proves a uniqueness result about their difference.

---

## Secondary Classifications

**11A05** -- Multiplicative structure; factorization

The proof relies on the multiplicativity of both sigma and tau, decomposing the problem into local factors at each prime power in the factorization of n.

**11N64** -- Other results on the distribution of values or the characterization of arithmetic functions

The result characterizes the value set {n : sigma(n) - tau(n) = 8} = {6}, which falls under the study of level sets of arithmetic functions.

**11B83** -- Special sequences and polynomials (for the Golay code connection, if included)
