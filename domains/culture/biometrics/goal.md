## 핵심 상수 매핑

```
n=6          : 모달 수 (지문/홍채/얼굴/정맥/음성/걸음)
tau=4        : 핵심 특징 축 (기하/텍스처/동역학/시계열)
sopfr=5      : 템플릿 필드 (ID/hash/timestamp/issuer/revoke)
n/phi=3      : 3-요소 인증 (지식/소유/생체)
phi=2        : 라이브니스 이중 검증
sigma=12     : ISO/IEC 생체 표준 카테고리
sigma-phi=10 : FAR 지수 (10^-10 목표)
J_2=24       : 연속 인증 주기 (시간)
```

## 5. 한계·MISS 정직 기록

- 걸음걸이는 부상/신발 영향 — tau=4 축 중 동역학 가변성
- 음성은 감기/소음 환경 — 라이브니스 φ=2 난이도↑
- 정맥 센서는 장비가 큼 — 웨어러블 통합 미해결
- 10⁻¹² FAR는 이론치, 현장 대규모 검증 필요
- ISO σ=12 표준 매핑은 카테고리 분할 기준에 따라 변동
