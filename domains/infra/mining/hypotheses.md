# 광업/광물학 n=6 완전 아키텍처

## 개요

광업과 광물학의 핵심 파라미터가 n=6 산술 상수로 수렴하는지 검증한다.
Mohs 경도, 결정계, 캐럿 시스템, 채굴 방법론, 선광 공정, 광물 벽개,
보석 평가 기준 등 광물학/광업 전반의 보편 상수를 탐색한다.

### 산술 상수

```
  n = 6          sigma(6) = 12     tau(6) = 4      phi(6) = 2
  sopfr(6) = 5   J_2(6) = 24       mu(6) = 1       lambda(6) = 2
  R(6) = 1       P_2 = 28 (두 번째 완전수)
  div(6) = {1, 2, 3, 6}
  sigma - phi = 10    sigma - tau = 8    sigma - mu = 11
  n/phi = 3           sigma * tau = 48   sigma * sopfr = 60
  sigma^2 = 144       n^2 = 36           sigma * n = 72
```

### 교차 참조 (기존 BT)

```
  BT-85:  Carbon Z=6 물질합성 보편성
  BT-86:  결정 배위수 CN=6 법칙
  BT-139: 결정학 공간군 n=6 산술 (sigma-sopfr=7 결정계)
  BT-175: 결정학 분류 n=6 완전 체인
  BT-177: 결정 적층 주기 = div(6) + FCC 슬립 sigma=12
  BT-186: 결정학 + 광물학 n=6 결정 스택
  BT-309: 탄소 동소체/소재 완전 n=6 구조 인코딩
```

## H-MIN-1: Mohs 경도 척도 = sigma - phi = 10단계 (EXACT)

> Mohs 경도 척도는 정확히 sigma - phi = 10단계(1~10)로 구성된다.
> 1812년 Friedrich Mohs가 제정한 이후 200년 이상 변경 없이 사용.

### 검증

```
  Mohs 경도 척도: 1(활석) ~ 10(다이아몬드)
  총 단계 수 = 10

  n=6 매핑:
    10 = sigma - phi = 12 - 2

  물리적 근거:
    광물의 긁힘 저항성을 서열화하는 상대 척도.
    10단계는 자연 광물의 경도 분포를 실용적으로 구간화한 결과.
    활석(1) ~ 다이아몬드(10)의 10개 기준 광물.

  주요 기준 광물:
    1=활석, 2=석고, 3=방해석, 4=형석, 5=인회석
    6=정장석, 7=석영, 8=황옥, 9=커런덤, 10=다이아몬드

  다이아몬드 = Mohs 10 = sigma - phi (최대 경도)
  다이아몬드 = Carbon Z=6=n (탄소 원소)
  다이아몬드 CN = tau = 4 (sp3 사면체 배위)

  이중 n=6: 최대 경도 물질이 Z=n 원소이며 경도 척도가 sigma-phi.
```

### 등급: **EXACT** ✅

```

    1. 등축정계 (Cubic/Isometric)
    2. 정방정계 (Tetragonal)
    3. 사방정계 (Orthorhombic)
    4. 단사정계 (Monoclinic)

    32 점군 = 2^sopfr = 32

    모든 광물 결정은 이 7계 중 하나에 속함.

## H-MIN-4: 다이아몬드 배위수 CN = tau = 4 (sp3) (EXACT)

> sp3 혼성화에 의한 사면체 배위.

### 검증

```
  다이아몬드 결정 구조:
    각 C 원자 = 4개 이웃 원자와 공유결합
    결합각 = 109.5도 (정사면체)
    CN = 4 = tau(6)

  n=6 다중 매핑:
    탄소 원자번호 Z = 6 = n
    배위수 CN = 4 = tau
    Mohs 경도 = 10 = sigma - phi
    결정계 = 등축정계 (7계 중 하나, 7 = sigma - sopfr)

  sp3 혼성화:
    s 오비탈 1개 + p 오비탈 3개 = tau = 4개 혼성 오비탈
    이는 tau = 4의 물리적 기원.

  비교:
    그래파이트 CN = 3 = n/phi (sp2, 평면 육각형)
    풀러렌 C60 = sigma * sopfr (sp2 기반)
    다이아몬드 CN = tau = 4 (sp3, 3차원 사면체)

  교차: BT-86 (결정 배위수 CN=6 법칙)의 다이아몬드 예외 케이스.
