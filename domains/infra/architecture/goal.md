```

tau=4        : 기본 구조 요소 (기둥/보/슬래브/전단벽)
sopfr=5      : 시공 공정 단계 (지반→골조→외피→설비→마감)
n/phi=3      : 환경 제어 축 (열/빛/공기)
phi=2        : 동선 위계 (공용/사적)
sigma=12     : 설계·시공 총 단계 (기획~준공)
sigma-phi=10 : 표준 마감재 종류
J_2=24       : 거주 사이클 (시간), 모듈 조립 단위(h)
```

## 5. 한계·MISS 정직 기록

- 공간 모듈 6종 매핑은 거주 유형 평균 — 1인/4인 가구에 따라 5~7
- 지진 내력 1.8g는 시뮬레이션 값, 실대형 진동대 미실증
- 수명 300년은 콘크리트 가속시험 외삽 + 부재 교체 전제
- 자연채광 90%는 위도 35~37° 기준, 고위도 미적용
- τ=4 가동축은 프로토타입 단계 (액추에이터 수명 미확정)
- σ=12 단계 매핑은 일반 건축 표준(기획·계획·기본·실시·발주·착공·골조·외피·설비·마감·시운전·준공) 준수
