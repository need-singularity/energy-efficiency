## 핵심 상수 매핑

```
n=6          : 좌표계 클래스(지심직교/지심구면/측지/지도투영/그리드/로컬)
tau=4        : 투영 패밀리(원통/원뿔/방위/하이브리드)
sopfr=5      : 측량 공정(기준점→관측→조정→변환→배포)
n/phi=3      : 공간 차원(점/선/면), 좌표축(X/Y/Z)
phi=2        : 측지/투영 이중성, 래스터/벡터 이중표현
sigma=12     : 공간분석 단계(수집→정제→투영→오버레이→버퍼→교차→
              → 근접→네트워크→통계→보간→시각화→배포)
sigma-phi=10 : 표준 레이어 종류(도로/건물/수계/식생/지형/행정/POI/
              → 인구/위험/유틸리티)
J_2=24       : 시간 단면(h, 일변동), 갱신 주기 시각
```

## 6. 한계·MISS 정직 기록

- 좌표계 클래스 6개는 ISO 19111 분류 압축 결과 (느슨)
- 투영 패밀리 4는 Snyder 분류와 일치 (강함)
- σ=12 분석 단계는 ESRI/QGIS 워크플로우 평균값
- 실내 0.8m는 UWB 기준 — BLE만으론 1.5m 한계
- 변화탐지 정확도 ML 의존, 도메인별 재학습 필요
