## 핵심 상수 매핑

```
n=6          : 힉스 붕괴 주 채널, 쿼크 플레이버, 세대×2
sigma=12     : SU(3)×SU(2)×U(1) 게이지 생성자 8+3+1
tau=4        : 전기약 게이지 보손 (γ,W+,W-,Z)
phi=2        : 힉스 더블릿 복소 성분
sopfr=5      : 힉스 포텐셜 다항식 차수 분해 2+3
J_2=24       : 유카와 결합 행렬 24원소 (CKM+PMNS)
sigma-phi=10 : 독립 자유 파라미터
sigma-tau=8  : SU(3) 글루온 8
sigma+phi=14 : SM 물리 세대 성분
sopfr^3=125  : 힉스 질량(GeV) ≈ 125
```
