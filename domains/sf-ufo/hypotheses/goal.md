## 5. BT(돌파 정리) 연결

- **BT-12 (NEXUS-6 합의)**: 3+ 렌즈 후보, 7+ 고신뢰, 12+ 확정 — σ=12 채널과 일치
- **BT-94~98 (CCUS 추론연쇄)**: 인과사슬 길이 ≤ σ=12 가설만 우선순위 상위
- **BT-344~346 (HEXA-GATE 오염방지)**: τ=4 게이트로 자기참조·표절 0
- **BT-401~408 (양자측정)**: 베이지안 갱신 = τ=4 측정사이클 모형
- **R1 정리 σφ=nτ**: 가설 평가식의 분모/분자에 직접 등장 (verify_alien10 참조)

## 6. Testable Predictions

1. **P-H1**: HEXA 파이프라인 가설의 사전등록 후 재현성이 무작위 LLM 가설 대비 n=6배 이상.
2. **P-H2**: 우선순위 상위 σ=12건 중 n=6건 이상이 12개월 내 1회 이상 인용.
3. **P-H3**: τ=4 게이트 도입 시 자기참조 비율이 단조 0으로 감소.
4. **P-H4**: sopfr=5 평가축 가중치를 균일화하면 사람 상관이 0.84→0.70 이하로 하락.
5. **P-H5**: 융합 가설(분야≥2) 비율이 (σ-φ)=10% 이상 유지될 때 BT 승격률 6배.
6. **P-H6**: 반증 시도 평균이 n=6 미만 가설군의 1년 폐기율이 6배 높음.

각 예측은 `docs/testable-predictions.md` 등재 예정.

## 7. 한계·MISS 정직 기록

- Ouroboros 자기참조 차수 n=6은 대부분 구조 검증, 무한 재귀 사례 3건 MISS
- σ=12 변환기 보편성은 자연어 영역 MISS

- 자동 NEXUS-6 렌즈 합의는 초신규 가설에서 후보단계 정체
- 검증코드 생성 자동화는 현재 수동 템플릿 기반
