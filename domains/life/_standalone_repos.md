## Convention

- Spec files in `domains/life/<slug>/<slug>.md` 은 canonical (uchg-seal). 편집 금지.
- Standalone 레포 (`github.com/need-singularity/*`) 는 작동 구현 + 시드 복사.
- README in each standalone 은 doc/ 시드를 통해 canonical로 역참조; 본 인덱스는 정참조.
