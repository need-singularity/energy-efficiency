## 핵심 상수 매핑

```
n=6          : 잠복 저수지 클래스, bNAb 에피토프 부위, ART 약물 클래스
tau=4        : gp120-CD4 접촉 핵심, RT 도메인, IN LEDGF 인터페이스
sopfr=5      : PR 활성부위 잔기, CCR5 ECL 구성
n/phi=3      : Tat-TAR 루프, Rev-RRE 스템, CCR5 ECL 수
phi=2        : PR 다이머, RT 이중 서브유닛
sigma=12     : FDA 승인 ART 단일제 계열
sigma-phi=10 : RT 오류율 지수 (10^-4 ~ 10^-5)
sigma-sopfr=7: IN LTR 인식 염기
sigma+phi=14 : bNAb 슈퍼사이트 인식 잔기
J_2=24       : 장기지속 주사 간격 (주)
```
