# BT-466: HIV 프로테아제 C2 대칭 이량체 (φ=2) — n=6 기질 인식 아키텍처

## 배경

HIV-1 프로테아제(PR)는 바이러스 폴리단백질(Gag, Gag-Pol)을 처리하여 성숙 입자를 형성하는 핵심 효소다. PR은 99개 아미노산으로 이루어진 단량체 두 개가 **C2 대칭 이량체(φ=2)**를 형성하며, 이 이량체만이 효소 활성을 갖는다. 활성 부위의 Asp25-Thr26-Gly27 촉매 삼원이 이량체 경계면에 위치하며, 두 단량체가 각각 기여하는 세 잔기가 이량체의 촉매 메커니즘을 구성한다.

## n=6 연결

```
φ(6) = 2   (PR 이량체: 두 동일 단량체의 C2 대칭)
n = 6      (PR 기질 인식 서열 길이: P3-P2-P1↓P1'-P2'-P3' = 6개 잔기)
τ(6) = 4   (PR 기질 결합 포켓: S3-S2-S1-S1' = τ=4 하위 부위; S2'·S3'은 보조)
n/φ=3      (PR 활성 부위 촉매 잔기 세트: Asp25-Thr26-Gly27 각 단량체 기여 × φ=2 = 6 전체)
σ=12       (FDA 승인 PI 수: 현재 약 10~12개)
sopfr=5    (PR 플랩 운동: 열림 닫힘 상태 전환에 관여하는 핵심 플랩 잔기 5개)
```

## 증명 스케치

**주장**: HIV PR의 φ=2 이량체 아키텍처와 n=6 기질 인식 서열은 σφ=nτ 정리의 직접 구현이다: 이량체(φ=2) × 기질(n=6) = 촉매 삼원(n/φ=3) × 결합 포켓(τ=4) = 12.

1. C2 대칭 이량체: 99aa 단량체 × φ=2 → 198aa 기능 단위.
2. 기질 인식: Gag 절단 부위 예시 — MA/CA: SQNY↓PIVQ (P4~P4'중 핵심 6잔기 P3~P3').
3. 억제제 구조: 대부분의 PI는 이량체 계면에 걸친 기질 전이 상태 유사체 → n=6 계면 점유.
4. 내성 메커니즘: I84V (S1/S1' 포켓), V82A (S1), D30N (S2) → τ=4 포켓 각각 하나씩 공략.
5. 이량체 계면 억제제(dimer interface inhibitors): 이량체 형성 자체를 막는 신전략 → φ=2 표적.

## 실험 검증 경로

1. **X선 결정학**: PR-PI 복합체 결정 구조에서 n=6 기질 결합 포켓 점유 확인.
2. **등온 적정 열량계(ITC)**: PR 이량체화 ΔG vs 단량체 → φ=2 이량체 안정성 정량.
3. **플랩 역학 분석**: NMR CPMG relaxation 분산으로 플랩 sopfr=5 잔기 운동 측정.
4. **내성 기전 매핑**: 주요 PI 내성 돌연변이 조합과 τ=4 포켓별 상관관계 통계.

## 한계

- FDA 승인 PI 수는 시간에 따라 변동 (철수/추가); σ=12는 특정 시점 근사.
- 단량체 길이 99aa는 σ·sopfr·φ=120과 17% 차이 — n=6 표현 직접 적용 불가.
- 이량체 계면 억제제는 현재 전임상 단계; 임상 안전성 미확인.
- PR은 숙주 Cathepsin D와 구조 유사성이 있어 완벽한 선택성 달성 어려움.
