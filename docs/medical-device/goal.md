## Core n=6 Anchors

```
  ECG (심전도):
    - 표준 리드:      σ = 12 leads EXACT
    - 사지 리드:      n = 6 limb leads EXACT
    - 흉부 리드:      n = 6 precordial leads EXACT
    - PR 간격:        120-200 ms = σ(σ-φ) ~ φ·100

  MRI (자기공명영상):
    - 코일 채널:      σ = 12 channels (standard)
    - 대조 시퀀스:    n = 6 sequences (T1/T2/PD/FLAIR/DWI/SWI)
    - 슬라이스:       J₂ = 24 slices

  Ultrasound (초음파):
    - 주파수:          n = 6 MHz (복부 표준)
    - 모드:            τ = 4 modes (B/M/D/Color)

  ADC/센서:
    - ADC 비트:        σ-τ = 8 bit (표준 의료)
    - 전극 배열:       σ = 12 (ECG EXACT)

  AI 진단 (BT-66 활용):
    - Vision AI:      ViT patch = σ-τ = 8 (radiology)
    - LLM:            BT-56 d = 2^σ = 4096 (medical text)
```
