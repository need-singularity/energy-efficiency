## 핵심 상수 매핑

```
n=6          : 주요 약리계(진통/진정/항염/항균/항산화/면역조절)
                또는 6대 활성기(알칼로이드/플라보노이드/테르페노이드/
                사포닌/페놀산/배당체)
tau=4        : 추출 4공정(분쇄 → 침출 → 분리 → 농축)
sopfr=5      : 조제 5단(원초 → 절단 → 포제 → 배합 → 환/탕/산/고/주)
n/phi=3      : 약성 삼분(寒/平/溫), 귀경 삼초(上/中/下)
phi=2        : 극성 용매/비극성 용매 이원 추출
sigma=12     : 표준화 12단계(정선→세척→건조→분쇄→체별→
                추출→여과→농축→건조→분말→배합→포장)
sigma-phi=10 : 허용 보조성분 종류
J_2=24       : 표준 침출 시간(h), 효능 검정 주기
```

## 5. 한계·MISS 정직 기록

- 식물 활성기는 실제로 8~10대 분류군 — n=6 강제 매핑은 주요계 한정
- 추출 4공정 중 "포제"는 한방 전통 단계로 τ=4 공식과는 별개 축
- 부작용 1/6 감소는 동물모델 예측치, 임상 미확보
- 잔류 용매 100 ppm은 에탄올 기준, 메탄올/헥산은 별도
- 귀경 삼초(上/中/下) 매핑은 전통 분류로 현대 약리와 직접 대응 불가
- 보존 36개월은 가속 안정성시험 외삽
- σ=12 표준화 단계는 GMP 기준 9~14단계 범위에서 n=6 친화 12로 고정
- 약리계 6분류는 WHO/한국약전 분류와 부분 일치 (4/6)
