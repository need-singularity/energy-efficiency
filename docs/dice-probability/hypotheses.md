# N6 주사위/확률/보드게임 -- 완전수 산술과 게임 구조 상수

## 개요

인류의 게임 도구 -- 주사위, 트럼프 카드, 마작, 바둑, 체스, 백개먼 --
의 구조적 상수를 n=6 산술함수로 분석한다. 게임은 이산적 계수
(면 수, 카드 수, 칸 수)가 정확히 고정되어 있어 수론적 검증에 이상적이다.

> **정직성 원칙**: 게임 규칙은 인간이 설계한 관습이다.
> 그러나 수천 년 지속된 보편적 구조(주사위 6면, 카드 52장)는
> "인간이 자연스럽게 선택한 이산적 최적"으로서 n=6 일치를 검증한다.
> 바둑 19×19처럼 직접 관계가 어려운 경우 솔직히 WEAK로 평가한다.

> 표준 주사위의 대면(반대편 면)의 합은 항상 σ-sopfr = 7

```

    1+6 = 2+5 = 3+4 = 7 = σ-sopfr = 12-5

  n=6 분해:

    서양, 동양 주사위 전부 대면 합 = 7
    (단, 동양 일부 주사위는 1,4가 빨간색 = φ색)

  등급: EXACT (수학적 필연, n+1 = σ-sopfr)
```
