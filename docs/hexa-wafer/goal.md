# 궁극의 웨이퍼 스케일 아키텍처 — HEXA-WAFER

> **Grade 참조**: alien_index = 제품 maturity (1~10). closure_grade = n=6 닫힘 등급 (1~13+).
> 현재: alien_index 7 maturity / closure_grade 6 (bt_exact_pct 기반 추정).

**Rating**: 7/10 -- 웨이퍼급 칩 설계에 n=6 산술 적용
**BT**: BT-90 (6D 패킹), BT-3 (sigma=12), BT-7 (Egyptian 분배)
**EXACT**: 28/36 (77.8%) -- 다이 배치, 수율, 인터커넥트
**DSE**: 2,985,984 조합 (6x24x36x48x24)
**Cross-DSE**: 반도체, 소재, 제조, 열관리, 패키징
**진화**: Mk.I(칩렛 기반)~V(물리한계 웨이퍼급 칩)
**불가능성 정리**: 9개 (수율~열밀도)

## Core Constants

```
n = 6          sigma(6) = 12     tau(6) = 4      phi(6) = 2
sopfr(6) = 5   J2(6) = 24        mu(6) = 1       lambda(6) = 2
R(6) = sigma*phi / (n*tau) = 1
Egyptian: 1/2 + 1/3 + 1/6 = 1

웨이퍼 특화:
웨이퍼 직경 = sigma = 12 인치 (300mm, 현행 표준, EXACT)
다이 타일 = sigma^2 = 144 (웨이퍼당 타일 수)
칩렛 수 = J2 = 24 (다이 당 칩렛 조합)
TSV 피치 = sopfr = 5 um (3D 적층 비아 피치)
메탈 층 = sigma = 12 (현행 첨단 공정 실측 13~15)
공정 노드 = tau = 4 (5nm, 3nm, 2nm, 1.4nm)
수율 목표 = sigma/(sigma+mu) = 92% (다이당)
전력 밀도 = J2-tau = 20 W/cm^2 (허용 상한 근방)
```

