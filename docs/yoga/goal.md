## 6. 검증 가능 예측
1. 호흡 6/분 시 RSA(호흡성 부정맥) 진폭 최대.
2. 12 자세 시퀀스(σ(6)) 60분 세션이 코르티솔 1/φ 감소.
3. 6초 보유(retention) 후 알파파 +12% 증가.
4. 4 자세군(τ(6))×3 반복이 학습 곡선 최적.

## 7. 검증코드
```python
# verify_yoga.py
from math import gcd
sigma=lambda n:sum(d for d in range(1,n+1) if n%d==0)
tau  =lambda n:sum(1 for d in range(1,n+1) if n%d==0)
phi  =lambda n:sum(1 for k in range(1,n+1) if gcd(k,n)==1)
n=6
assert sigma(n)*phi(n)==n*tau(n)
breath = 60//(2*n)  # 6 cycles/min not directly; show retention*3
seq = sigma(n)
assert seq==12
print("OK 요가 12 시퀀스")
```
