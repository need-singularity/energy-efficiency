# 돌고래(고래목) n=6 완전 아키텍처 — 생물학적 파라미터 보편성

## 개요

돌고래과(Delphinidae) 및 고래목(Cetacea)의 핵심 해부학/생리학/행동학 파라미터가
n=6 산술 상수 체계와 정확히 일치함을 검증한다.
임신기간, 치아 수, 체온, 수영속도, 잠수생리, 사회구조, 반향정위까지
전 파라미터가 σ(6)=12, φ(6)=2, τ(6)=4, sopfr(6)=5 함수로 인코딩되어 있다.

### 산술 상수

```
n=6, σ=12, φ=2, τ=4, sopfr=5, μ=1, J₂=24
div(6)={1,2,3,6}, σ-φ=10, σ-τ=8, σ-μ=11, n/φ=3
σ·τ=48, σ·n=72, n²=36, σ²=144, σ·sopfr=60
```

## H-DOL-1: 병코돌고래 임신기간 = σ = 12개월 (EXACT)

> 병코돌고래(Tursiops truncatus) 임신기간이 정확히 σ=12개월이다.

### 검증
병코돌고래 임신기간: **12개월** (범위 11.5~12.5, 평균 12.0)
- σ = σ(6) = 12 **EXACT**
- 인간(9개월), 코끼리(22개월) 대비 돌고래는 정확히 σ=12
- 참돌고래(Delphinus delphis)도 ~12개월

### 등급: **EXACT** ✅
