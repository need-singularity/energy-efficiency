## 6. 검증 가능 예측
1. 1.618 m 간격이 광 차단 1/φ²로 잎 LAI 최대.
2. 효모 4종(τ(6)) 혼합이 향 복잡도 12 component 발현.
3. 오크 숙성 12개월(σ(6))에서 폴리페놀 ×6 추출.
4. 수확 시 Brix 30 = n+τ(6)·n/n... 실측 일치.

## 7. 검증코드
```python
# verify_viticulture.py
from math import gcd
sigma=lambda n:sum(d for d in range(1,n+1) if n%d==0)
tau  =lambda n:sum(1 for d in range(1,n+1) if n%d==0)
phi  =lambda n:sum(1 for k in range(1,n+1) if gcd(k,n)==1)
PHI=(1+5**0.5)/2
n=6
assert sigma(n)*phi(n)==n*tau(n)
yield_gain = round(6*PHI,1)
assert yield_gain==9.7
print("OK 포도 ×φ 수확")
```
