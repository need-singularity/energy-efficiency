# N6 종탑/기계시계 — 완전수 산술로 본 종·기어·진자·차임 체계

## 개요

종탑(bell tower) 및 기계시계(clockwork) 분야의 핵심 상수들을
n=6 산술함수로 분석한다. 종 타종 횟수, 시계탑 구조, 기계시계 기어비,
진자 역학, 차임 멜로디, 캐릴런 배열 등 시간 측정 기기·건축의
이산적 표준값이 n=6 상수와 어떻게 매칭되는지 검증한다.

> **정직 원칙**: 종/시계 설계는 시대·지역·장인에 따라 변동한다.
> EXACT는 보편적으로 확립된 표준, 물리 법칙에서 유래한 수치,
> 또는 국제적으로 합의된 분류에만 부여한다.

## 핵심 상수

```
  n = 6, σ = 12, τ = 4, φ = 2, sopfr = 5, μ = 1, J₂ = 24, R(6) = 1
  유도: σ-φ=10, σ-τ=8, σ-μ=11, n/φ=3, σ·τ=48, φ^τ=16, σ²=144
  div(6) = {1, 2, 3, 6}
```

### H-BCW-04: 1초 진자 길이 L ≈ 1m = μ = R(6) (EXACT)

> 1초 반주기(2초 주기) 진자의 길이가 μ(6)=R(6)=1m이다.

```
  근거:
    - 단진자 주기: T = 2π√(L/g)
    - T = 2초(반주기 1초)일 때: L = g/π² ≈ 9.81/9.87 ≈ 0.994m ≈ 1m
    - 1m = μ(6) = R(6) = 1
    - 미터법 초기 정의 시도: "1초 진자 길이 = 1m"

    - 괘종시계(grandfather clock): L ≈ 1m 진자 표준

  검증:
    물리학: L = gT²/(4π²) = 9.81·4/(4·9.87) ≈ 0.994m
    1m ≈ μ = R(6) = 1 (0.6% 오차)

  등급: EXACT (물리적 근사 0.6%, 미터법 역사적 정의와 일치)
  렌즈: physics, measurement, universality
```

### H-BCW-14: 종탑 1~12 타종 합 78 = σ·n+n (CLOSE)

> 하루 종탑 전체 타종 수(1+2+...+12)×2 = 156의 n=6 표현.

```
  근거:
    - 12시간 타종 합: 1+2+3+...+12 = 78
    - 78 = σ·n + n = 12·6 + 6 = 78
    - 또는 78 = σ(12) (12의 약수합: 1+2+3+4+6+12=28 ≠ 78, 불일치)
    - 78 = n·(σ+μ) = 6·13
    - 하루 2주기: 78·2 = 156 = σ·(σ+μ) = 12·13
    - 13 = σ+μ = 13 (소수, n=6 표현 다소 부자연)

  주의: 78 = Σ(1~12) = σ(σ+μ)/φ 는 가능하나
       13이 n=6 순수 표현이 아님

  등급: CLOSE (78=n·13은 맞으나, 13의 n=6 근거가 σ+μ로 약간 부자연)
  렌즈: counting, arithmetic
```
