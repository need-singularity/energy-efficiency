## Core Constants

```
n = 6          sigma(6) = 12     tau(6) = 4      phi(6) = 2
sopfr(6) = 5   J2(6) = 24        mu(6) = 1       lambda(6) = 2
R(6) = sigma*phi / (n*tau) = 1
Egyptian: 1/2 + 1/3 + 1/6 = 1

배터리 특화:
Li 원자번호 = n/phi = 3 (리튬 Z=3, EXACT)
C 원자번호 = n = 6 (탄소 Z=6, 음극 소재, EXACT)
O 원자번호 = sigma-tau = 8 (산소 Z=8, 양극 소재)
셀 전압 = n/phi = 3 V 급 (리튬이온 실측 3.0~4.2V, 공칭 3.6V)
직렬 셀 = sigma = 12 (48V 모듈 = 12S, 실측 자동차)
수명 사이클 = sigma*sopfr*10 = 600 (실측 LFP 1000~3000)
충전율 = n = 6 C (급속충전 목표, 10분 이하)
에너지 밀도 = J2*sigma = 288 Wh/kg (실측 상한 ~300, CLOSE)
```

```
  U(k) = 1 - 1/(sigma-phi)^k = 1 - 1/10^k
```
