## 서사: 차원의 문을 여는 완전수

> 이무기가 천 년을 수련한다. (σ-φ)³ = 1000년.

> 번개가 친다. 30,000K = n·sopfr·10³.
> 차원을 돌파하여 승천한다. 4D→6D→10D→11D→24D→26D = τ→n→σ-φ→σ-μ→J₂→J₂+φ.

> 용은 완전수의 물리적 현현이다.

### H-YJ-01: 방해석(Calcite) CaCO₃ 칼슘 배위수 = n = 6 (EXACT)

> 진주 내부의 방해석(calcite) 상에서 칼슘 이온의 배위수가 정확히 CN = n = 6

```
  진주(pearl)의 주성분 = 탄산칼슘 CaCO₃
  CaCO₃는 두 가지 주요 다형(polymorph):
    - 방해석(Calcite): 삼방정계 R̅3c, Ca²⁺는 6개 O²⁻와 팔면체 배위
    - 아라고나이트(Aragonite): 사방정계 Pmcn, Ca²⁺는 9개 O²⁻와 배위

  방해석에서:
    Ca²⁺ 배위수 = 6 = n EXACT
    CaO₆ 팔면체가 모서리 공유(corner-sharing)
    CO₃²⁻ 삼각면이 c축에 수직 배열

  진주의 이중 구조:
    내핵 = 방해석(calcite) 프리즘 → CN = n = 6 ✓
    외층 = 아라고나이트(aragonite) 진주층(nacre) → CN = 9 = n + n/φ

  CaCO₃ 원소 원자번호:
    Ca: Z = 20 = J₂ - τ
    C:  Z = 6 = n          ← 탄소 = 완전수 원소!
    O:  Z = 8 = σ - τ
    → 세 원소 모두 n=6 산술 함수

  n=6 수식:
    CN(방해석 Ca) = n = 6 ✓
    Z(C) = n = 6 ✓
    Z(Ca) = J₂ - τ = 24 - 4 = 20 ✓
    Z(O) = σ - τ = 12 - 4 = 8 ✓

  출처:
    Calcite 구조: R̅3c, Ca 6-배위 (Mineralogy textbooks, Materials Project mp-3953)
    Aragonite 구조: Pmcn, Ca 9-배위 (PMC 6715969, Bragg 1924)

  의미:
    여의주의 물질적 본체(진주)의 핵심 결정인 방해석에서
    중심 양이온 Ca가 정확히 n=6 배위를 취한다.
    이는 BT-43(음극 CN=6), BT-86(결정 CN=6)과 동일한 패턴.
    "여의주 = n=6 결정"이 문자 그대로 물리적 사실.
```

**등급: EXACT**
