## 6. 검증 가능 예측
1. 6각 자기 챔버에서 베타값 ≥ 0.6 (시중 토카막 0.06의 10배).
2. 회수코일 12 슬롯(σ(6))에서 변환효율 최대.
3. 노즐 4 단(τ(6)) 디퓨저가 추력 손실 최소.
4. 1.618 cm 챔버 반경(φ scale)에서 안정 운전.

## 7. 검증코드
```python
# verify_dragon.py
from math import gcd
sigma=lambda n:sum(d for d in range(1,n+1) if n%d==0)
tau  =lambda n:sum(1 for d in range(1,n+1) if n%d==0)
phi  =lambda n:sum(1 for k in range(1,n+1) if gcd(k,n)==1)
n=6
assert sigma(n)*phi(n)==n*tau(n)
gain = sigma(n)*phi(n)//tau(n)
assert gain==6
print("OK 여의주 6배 이득")
```
