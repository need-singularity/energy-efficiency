## 5. 한계·MISS 정직 기록

- DDC/KDC 실제 4단 강제 — 5~6단 세분류 라이브러리 존재 (τ=4 강매핑은 근사)
- σ=12 보존 단계는 OAIS 모델(6기능)을 12단으로 세분한 설계치 — 표준 미합의
- 디지털 300년 수명 — 마이그레이션 자동화 가정, 실측 외삽
- n=6 메타축은 Dublin Core 15요소 / MARC 200+필드 압축 — 무손실 검증 필요
- 다국어 72는 ISO 639-1 184개 중 운용 우선순위, σ × n 산식은 설명적
