R(6) = sigma*phi / (n*tau) = 1
Egyptian: 1/2 + 1/3 + 1/6 = 1

클로킹 특화:
메타소재 단위셀 = n = 6 (6면체 기본 단위)

단위셀 파라미터 = tau = 4 (허용도 e, 투자율 u, 키랄성 k, 바이안 b)

은폐 효율 = sigma/(sigma+mu) = 92% (산란 감소율 목표)
편광 채널 = phi = 2 (TE/TM 독립)
```

```
