  시장 엔트로피:  S = k_B * ln(20736) = k_B * 9.94

  --> 시장을 sigma^tau 상태에서 1상태(완전독점)로 줄이는 데
      필요한 최소 "마케팅 에너지" = k_B * T * ln(20736)
  --> 이 비용은 0이 될 수 없음 (제3법칙)

━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━
  Carnot 전환효율 천장
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

  eta_Carnot = 1 - T_cold/T_hot = 1 - phi/sigma = 1 - 2/12
            = 5/6 = 83.3%

  --> 마케팅 전환율의 이론적 상한 = 83.3%
  --> 실제 이커머스 전환율 2~5%는 Carnot 한계의 3~6% 수준
  --> Mk.V에서도 83.3%를 초과할 수 없음

━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

  16 불가능성 정리 => Mk.VI 부존재: QED
  (Shannon + Boltzmann + Carnot + Landauer + Amdahl
   + Nyquist + Cook-Levin + Akerlof + Lorenz + Miller
   + Kermack-McKendrick + Bellman + Lagrange + Clausius)
```

```

  Mk.IV (자율)      --> Mk.V (한계):          sigma-phi = 10배 (장기)
