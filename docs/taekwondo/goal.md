## 6. 검증 가능 예측
1. 골반 회전 φ rad/s에서 발끝 속도 최대.
2. 4 기본자세(τ(6))×3 반복이 안정성 σ(6)=12배.
3. 발차기 임팩트 6 ms 지속이 격파력 최대화.
4. 12 콤보(σ(6))가 경기 점수 평균 +6점.

## 7. 검증코드
```python
# verify_taekwondo.py
from math import gcd
sigma=lambda n:sum(d for d in range(1,n+1) if n%d==0)
tau  =lambda n:sum(1 for d in range(1,n+1) if n%d==0)
phi  =lambda n:sum(1 for k in range(1,n+1) if gcd(k,n)==1)
PHI=(1+5**0.5)/2
n=6
assert sigma(n)*phi(n)==n*tau(n)
speed = round(6*PHI,1)
assert speed==9.7
print("OK 발차기 ×φ")
```
