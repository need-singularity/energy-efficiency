## 6. 상세 가설 검증

### H-WINE-1: 알코올 발효 화학양론 — J₂=24 원자 완전 보존

- **발견**: 와인 발효의 핵심 반응 C₆H₁₂O₆→2C₂H₅OH+2CO₂에서 총원자수 J₂=24 보존
- **수식**: 반응물 24원자(6C+12H+6O) = 생성물 24원자 = J₂(6)
- **검증**: Gay-Lussac(1810) 방정식, 질량 보존 법칙에 의해 불변. BT-27, BT-101 직접 연결
- **등급**: EXACT

### H-WINE-2: 와인 평가 6대 요소 — n=6

- **발견**: 국제 소믈리에 교육(WSET, Court of Master Sommeliers)의 관능 평가 기본 요소가 6개
- **수식**: 평가 요소 수 = n = 6
- **검증**: (1)산도 (2)당도 (3)타닌 (4)알코올 (5)바디 (6)아로마. WSET Level 3+ 체계적 시음 접근법(SAT) 표준
- **등급**: EXACT

### H-WINE-3: 보르도 블렌드 6대 품종 — n=6

- **발견**: 보르도 AOC 규정에서 허용하는 레드 와인 주요 포도 품종이 n=6개
- **수식**: 품종 수 = n = 6
- **검증**: 카베르네 소비뇽/메를로/카베르네 프랑/프티 베르도/말벡/카르메네르. 보르도 AOC 규정 확인. 일부 문헌에서 카르메네르를 제외하나 공식 허용 품종은 6개
- **등급**: EXACT

### H-WINE-4: 오크 숙성 표준 12개월 — σ=12

- **발견**: 와인 오크 배럴 숙성 표준 기간이 σ=12개월
- **수식**: 숙성 기간 = σ = 12개월
- **검증**: 스페인 Reserva 기준 오크 12개월 최소, 이탈리아 DOCG 유사 규정. 225L 바리크(보르도)/228L 피에스(부르고뉴)에서 12개월이 표준 1년 숙성
- **등급**: EXACT

### H-WINE-5: 레드와인 서빙 온도 — σ=12°C

- **발견**: 가벼운 레드와인(피노 누아, 가메이 등)의 최적 서빙 온도가 σ=12°C
- **수식**: 서빙 온도 = σ = 12°C
- **검증**: WSET 권장: 라이트 레드 12~14°C. 보졸레 누보 12°C, 피노 누아 12~14°C. 풀 바디 레드는 16~18°C=φ^τ~σ+n
- **등급**: EXACT

### H-WINE-6: 부르고뉴 4등급 — τ=4

- **발견**: 부르고뉴 와인의 품질 등급 체계가 τ=4단계
- **수식**: 등급 수 = τ = 4
- **검증**: (1)Regional (2)Village (3)Premier Cru (4)Grand Cru. INAO/AOC 공식 분류, 1930년대 확정 이래 불변
- **등급**: EXACT

### H-WINE-7: 보르도 1855 분류 5등급 — sopfr=5

- **발견**: 1855년 보르도 메독 와인 공식 분류가 sopfr=5개 등급
- **수식**: 등급 수 = sopfr = 5
- **검증**: Premier Cru(1등급)~Cinquieme Cru(5등급). 나폴레옹 3세 파리 만국박람회 위한 분류, 1855년 이래 Mouton Rothschild 승격(1973) 제외 불변
- **등급**: EXACT

### H-WINE-8: 에탄올 탄소 수 — φ=2

- **발견**: 에탄올 C₂H₅OH의 탄소 골격이 φ=2개
- **수식**: C₂ → 탄소 수 = φ = 2
- **검증**: 에탄올은 2탄소 알코올. 화학적 사실
- **등급**: EXACT

### H-WINE-9: 발효 생성물 계수 — φ=2

- **발견**: 포도당 1분자에서 에탄올 φ=2분자, CO₂ φ=2분자 생성
- **수식**: 생성물 계수 = φ = 2 (양쪽 동일)
- **검증**: C₆H₁₂O₆ → **2**C₂H₅OH + **2**CO₂. 화학양론 불변
- **등급**: EXACT

### H-WINE-10: 테이스팅 아로마 3계층 — n/φ=3

- **발견**: 와인 아로마가 n/φ=3 계층으로 분류됨
- **수식**: 아로마 계층 = n/φ = 3
- **검증**: (1)1차 아로마: 포도 품종 고유 향 (2)2차 아로마: 발효 과정 생성 향 (3)3차 아로마(부케): 숙성 과정 발달 향. WSET/CMS 표준 분류
- **등급**: EXACT
