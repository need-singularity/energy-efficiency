## 핵심 상수 매핑

```
n=6          : 표준 생물부품 클래스(프로모터/RBS/CDS/터미네이터/스캐폴드/절연체)
              균주 섀시(E.coli/효모/CHO/B.subtilis/P.pastoris/Synechocystis)
tau=4        : 논리 게이트(AND/OR/NOT/NAND), DNA 염기, 회로 상태
sopfr=5      : CRISPR Cas 클래스, AAV 세로타입 주류, 번역 단계
n/phi=3      : 중심교리(DNA→RNA→단백질), 코돈 3염기
phi=2        : 이중나선, 프로모터 강/약
sigma=12     : TCA 회로 단계 근사, 아미노산 대사군
sigma-phi=10 : 해당과정 효소 단계 (공유)
J_2=24       : mRNA 반감기(시간) 상한 근사
```
