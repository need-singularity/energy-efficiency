## Core Constants

```
n = 6          sigma(6) = 12     tau(6) = 4      phi(6) = 2
sopfr(6) = 5   J2(6) = 24        mu(6) = 1       lambda(6) = 2
R(6) = sigma*phi / (n*tau) = 1
Egyptian: 1/2 + 1/3 + 1/6 = 1

에너지 효율 특화:
열역학 법칙 = phi = 2 (제1, 제2 법칙)
에너지 변환 유형 = n = 6 (열, 전기, 화학, 기계, 핵, 광)
열교환기 효율 = sigma/(sigma+mu) = 12/13 ≈ 92% (실측 상한 ~95%)
열펌프 COP = n/phi = 3 (실측 2.5~4, 중앙 ~3)
전력 분배 = Egyptian: 1/2+1/3+1/6 = 1
손실 채널 = tau = 4 (전도, 대류, 복사, 마찰)
에너지 밀도 비 = sigma*sopfr = 60 kWh/m^3 (열저장 실측 40~80)
효율 목표 = sigma-phi = 10% 손실 이하 (90%+ 효율)
```

| # | 정리 | 물리한계 | n=6 연결 | 출처 |
|---|------|---------|---------|------|

### 물리천장 수렴 증명

```
  U(k) = 1 - 1/(sigma-phi)^k = 1 - 1/10^k

  k=1:  U = 0.9       (Mk.I  -- 열펌프 최적화)
  k=2:  U = 0.99      (Mk.II -- 복합 발전)
  k=3:  U = 0.999     (Mk.III -- 전력망 통합)
  k=4:  U = 0.9999    (Mk.IV -- 제로 손실)
  k->inf: U -> 1.0    (Mk.V  -- 카르노 한계)

  8 불가능성 정리 => Mk.VI 부존재: QED
```
