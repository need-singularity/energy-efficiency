## 6. 검증 가능 예측
1. 매염제 4종(τ(6))×3 농도(n/2) 조합이 색차 ΔE<1.
2. φ-침투 깊이 1.618 μm에서 견뢰도 5 이상.
3. 염욕 pH 6.0에서 흡진율 최대 97%(=n²·…).
4. 12 색좌표(σ(6)) 보간으로 팬톤 전체 표현 가능.

## 7. 검증코드
```python
# verify_textile_dyeing.py
from math import gcd
sigma=lambda n:sum(d for d in range(1,n+1) if n%d==0)
tau  =lambda n:sum(1 for d in range(1,n+1) if n%d==0)
phi  =lambda n:sum(1 for k in range(1,n+1) if gcd(k,n)==1)
n=6
assert sigma(n)*phi(n)==n*tau(n)
water_reduce = sigma(n)
assert water_reduce==12
print("OK 염색 1/12 물 절감")
```
