## 5. 검증 가능 예측 (요약)

1. **KATRIN 후속**: 5년 누적 시 |m²(ν_e)| < 4×10⁻² eV² (τ=4 계수)
2. **IceCube**: 1 PeV 이상 ν 도착 시각 분포 superluminal tail < 6σ (n=6)
3. **SN1987A 재해석**: ν와 광자 도착 차이 Δv/c < 2×10⁻⁹ (φ=2)
4. **LHC**: 13.6 TeV pp에서 진공-Cherenkov 유사 단면적 < 1/(σ=12) fb
5. **CMB 분광**: μ-distortion 한계 내 타키온 응축 기여 < 1/n=6
6. **GW 위상**: O4 데이터 분산 관계 위반 < 10⁻¹⁰ (sopfr=5 σ)
7. **저에너지 광학**: 군속도 > c 매질에서 신호속도 ≤ c 재확인 (n=6 매질)
8. **Higgs 부문**: 자체결합 λ 부호 변화점에서 m² < 0 영역 곡률 매핑
9. **우주론**: 진공 붕괴 시간 τ_vac > 10¹⁰⁰ 년 (n=6 채널 보정)
10. **인과율**: 폐쇄시간곡선(CTC) 형성 상한 < 10⁻⁴⁰ (σ-φ=10)

## 6. 한계·MISS 정직 기록

- 타키온은 **현재까지 직접 관측된 적 없다**. 본 연구는 "발견"이 아니라 "상한 압축"을 목표로 한다.
- 표준모형 내 "타키온 응축"(Higgs)은 m² < 0를 사용하지만 입자가 아니라 장의 불안정성이다 — n=6 매핑은 비유적 확장.
- KATRIN의 m² < 0 중심값은 통계적 요동일 가능성이 압도적. SF 해석 금지.
- "초광속 신호 불가능" (Lorentz 불변 + 양에너지 조건)은 이미 정리. 본 문서는 이를 뒤집지 않는다.
- n=6 채널, τ=4 기둥, σ=12 단계 매핑은 분석 프레임의 강제 정합이며 자연이 그렇게 분할된다는 주장이 아니다.
- 9번 항목 진공 안정성은 m_H, m_t 측정 의존, 모형 의존성 존재.
