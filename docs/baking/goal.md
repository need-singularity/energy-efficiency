## 핵심 상수 매핑

```
n=6          : 재료 6종(밀가루/물/이스트/소금/지방/당), 반죽 단계
tau=4        : 발효 단계(1차/펀치/벤치/2차), 글루텐 발달 페이즈
sopfr=5      : 굽기 온도 구간(예열/초기팽창/색깔/내부/식힘)
n/phi=3      : 효모 활성 주상(지수/대수/정체), 풍미 분자군
phi=2        : 수화/산화 주경로, 반죽 수분율 임계
sigma=12     : 전체 공정(계량→믹싱→1차→펀치→분할→벤치→성형→2차→스코어→굽기→식힘→포장)
sigma-phi=10 : 허용 첨가 부재료 종류
J_2=24       : 저온 숙성 기준시간(h), 풍미 발달 측정 주기
```
