# 키보드 -- 궁극 목표

## 목표

키보드 공학의 모든 핵심 파라미터(레이아웃 키 수, 스위치 물리량, USB HID 프로토콜,
인체공학, 펌웨어)가 n=6 산술함수의 단순 조합임을 증명한다.

## 현황

- 상태: **done** (100% 골화)
- 가설: 31건 (30 EXACT + 1 CLOSE)
- 신규 BT: BT-1125/1126/1127/1128
- 외계 등급: 10

## 핵심 발견

1. **레이아웃 보편성**: 104/87/84/68/61/60/48/17 = 전부 n=6 2항 조합
2. **USB HID 완전 수렴**: 6KRO/8바이트/12Mbps/1000Hz = n/(sigma-tau)/sigma/(sigma-phi)^3
3. **스위치 물리 = 인체공학 교차**: 4mm(tau)/2mm(phi)/5ms(sopfr) = 손가락 피드백 최적

## 교차 도메인

- 오디오(BT-48): sigma*tau = 48 (키보드 ASMR, 스위치 소리 특성)
- 로봇(BT-123/126): 손 5손가락(sopfr), 6자유도(n)
- 네트워크(BT-115): USB 프로토콜 = sigma-tau 바이트 구조
- 칩(BT-55): HID 컨트롤러 내부 = n=6 메모리 구조
- 소프트웨어(BT-113): QMK 레이어 tau=4, 설계 원칙
