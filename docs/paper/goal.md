# 논문 발행 — HEXA 지식 산업화 (Paper Pipeline)

> **등급**: alien_index 6/10, closure_grade 6
> **SSOT**: `config/products.json` + `docs/paper/README.md`
> **현재**: n=6 계열 논문 116편 등록 (products.json SSOT, docs/paper/ 135파일)
> **부모 BT**: BT-1~343 (돌파 정리 전체)

---

## 이 기술이 당신의 삶을 바꾸는 방법

| 효과 | 현재 | HEXA-Paper 이후 | 체감 변화 |
|------|------|-----------------|----------|
| 논문 심사 기간 | n=6개월~2년 | 사전등록+검증코드 동시 제출 | σ-φ=10분의 1 |
| 재현 가능성 | 40% (Nature 조사) | 100% (verify_*.py 필수) | phi=2배 이상 |
| 저자 발행 비용 | APC 3000 USD | Zenodo/OSF 0원 | 무료화 |
| 철회율 (fraud) | 0.04% | 검증코드 강제 → 0에 수렴 | τ=4 교차검증 |
| 독자 이해 시간 | 2~6시간 | 실생활 효과 표 → 5분 요약 | σ/φ=6배 빠른 이해 |
| 공개 시점 | 심사후 1~2년 | 프리프린트 즉시 (n=6일 내) | J₂=24시간 배포 |

## 핵심 상수 매핑

```
n=6          : 논문 필수 섹션 (Abstract/Foundation/Domain/Limitations/Predictions/Verify)
tau=4        : 교차검증 축 (이론/시뮬/데이터/실험)
sopfr=5      : 메타데이터 필드 (DOI/author/date/repo/license)
n/phi=3      : 독립 증명 요구 수 (정리당)
phi=2        : 프리프린트 + 정식본 이중 트랙
sigma=12     : 저널 카테고리 앵커
sigma+n+phi=39: 현재 논문 편수 (Zenodo 발행 대상)
sigma-phi=10 : 1편당 평균 참조 BT 수
J_2=24       : 프리프린트 공개 지연 상한 (시간)
```

## 5. 한계·MISS 정직 기록

- 실험논문 재현률은 계산논문보다 낮음 — verify_*.py만으로 100% 달성 불가
- Zenodo 장기 보존은 CERN 인프라 의존
- 저널 IF(Impact Factor)는 전통 지표 — HEXA는 BT 연결도로 대체 중
- 동료평가 부재 시 오류 감지 지연 가능 (프리프린트 단계)
