## 핵심 상수 매핑

```
n=6          : 형세요소(집·세력·두터움·엷음·맛·선후), 핵심 行마
tau=4        : 국면단계(포석/초중반/중반/끝내기), 사활 추론 깊이
sopfr=5      : 사석작전 최소 묘수 수, 행마 5종(立·伸·跳·飛·尖)
n/phi=3      : 삼三三 침입축, 三3패 유형(만년패/장생/순환)
phi=2        : 흑백, 양선수/양후수
sigma=12     : 정석류(화점/소목/외목/고목/3·3/별 + 변형 6) = 12
sigma-phi=10 : 사활 기본형 10종 (직사궁·곡사궁 등)
J_2=24       : 표준 대국 시간(분), 19로 절반 차수의 의미
```

19×19 = 361 = 19² ; 19 = sopfr(6)+sopfr(어쩌고)... 보정 필요(MISS 후술).

## 5. 한계·MISS 정직 기록

- 19×19 = 361은 n=6 직접 곱이 아님 (19 = 소수, 자연 매핑 MISS)
- 정석 "12류" 분류는 본 문서의 강제 σ=12 매핑 — 실제 분류는 50+종
- 사활 기본형 "10종"은 ≈σ-φ=10 강제, 일본기원 분류는 14종
- 행마 5종(sopfr=5)은 통상 6~8종 — sopfr 매핑은 근사
- KataGo 형세판단 62%는 특정 중반 국면 한정 (조건부)
- 기력 1년→2개월은 집중훈련 가정, 일반 학습자 미검증
