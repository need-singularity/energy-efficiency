# 도메인 3: CCR2 길항제 국소 제형

> 성숙도 30% | 5~10년 | 혁신성 100% (세계 최초, 논문 0건)

## 핵심 통찰

Baranska 2018이 확인한 사실: **새 대식세포는 CCR2 의존적으로 혈중 Ly-6C(hi) 단핵구에서 분화한다.**

→ CCR2를 막으면 보급선이 끊긴다.
→ 기존 대식세포가 죽어도 후속 부대가 오지 않는다.
→ 자유 잉크가 재포획 없이 림프관으로 배출.

## CCR2 vs CSF1R — 왜 CCR2가 맞는가

| 항목 | CCR2 길항 | CSF1R 억제 |
|------|-----------|-----------|
| 작용 대상 | 이동중 단핵구 | 모든 조직 대식세포 |
| 기존 대식세포 | 유지 | 90% 사살 |
| 면역 방어 | 대부분 유지 | 심각하게 저하 |
| 간독성 | 경미 (두통 8%) | 심각 (6.1%) |
| 복원 시간 | 중단 후 즉시 | 10~20일 |
| 비유 | 보급선 차단 | 핵폭격 |

CCR2 = 외과적 정밀 타격. CSF1R = 카펫 폭격.
문신 제거는 전쟁이 아니라 수술이다.

## 약물 후보

### 1순위: Cenicriviroc (CCR2/CCR5 이중 길항)

- 원 적응증: NASH (비알코올성 지방간)
- 임상 상태: 2상 완료 (CENTAUR), 3상 중단 (AURORA)
- 인간 안전성: 수백 명 데이터
- 부작용: 두통 11%, 오심 8%, 설사 7% (경미)
- 간독성: CSF1R 대비 극히 낮음
- 특허: Tobira → Allergan → AbbVie
- 경구 제형만 존재, 국소 없음

### 2순위: PF-04136309 (선택적 CCR2)

- 원 적응증: 췌장암 (Pfizer)
- 임상: 1상 완료
- 장점: CCR5 영향 없음
- 단점: 개발 중단 상태

### 전임상 도구: RS-102895

- 연구용 시약, 광범위 사용
- 인간 안전성 없음
- 가격 저렴, PoC 실험에 최적

## 국소 제형 설계

경구 → 국소 전환 이유: 전신 면역 영향 최소화

| 전달 방식 | 현실성 | 장점 |
|-----------|--------|------|
| 마이크로니들 패치 | 가장 현실적 | 도메인 2와 동일 플랫폼 |
| PLGA 나노입자 피내 주사 | 높음 | 1회 주사 → 7일 서방 |
| 이온토포레시스 겔 | 보조적 | 전기 보조 투과 |
| 리포솜 겔 | 낮음 | 진피 도달 어려움 |

최적 조합: PLGA 나노입자에 CCR2 길항제 봉입 → 피내 주사 1회 → 7일 서방 → 골든 윈도우 완전 커버

## 투여 타이밍

```
D-7: CCR2 길항제 투여 시작 (경구 또는 피내 주사)
     → 혈중/진피 단핵구 CCR2 포화
D+0: 레이저 시술
     → 잉크 방출 순간 보급선이 이미 끊겨 있음
D+3: 골든 윈도우 종료
D+7: 약물 중단 → 면역 즉시 복원
```

## 특허 전략

### 선행기술 조사

- Baranska 2018: 대식세포 사이클 규명 (학술, 비특허)
- 클로드로네이트 리포솜: 제네릭 (특허 만료)
- CCR2 길항제 문신 적용: **논문 0건, 특허 0건**
- → **신규성 확보 가능**

### 출원 가능 청구항

1. CCR2 길항제를 문신 부위에 국소 투여하여 레이저 문신 제거 효율을 향상시키는 방법
2. 클로드로네이트 리포솜과 CCR2 길항제를 순차 투여 후 레이저 조사하는 문신 제거 방법
3. 마이크로니들 패치에 대식세포 표적 약물을 탑재한 문신 제거용 의약품 전달 장치

### 비용 및 시점

- PCT 국제출원: 500~1,000만원
- 시점: **지금 즉시** (전임상 데이터 없어도 방법론 특허 출원 가능)
- 데이터 보강 시 등록 확률 상승

## 이것이 열어주는 것

1. Fitzpatrick V~VI: 레이저 에너지 낮춰도 면역 차단이 보상
2. 초록/파랑: 대식세포 사살로 물리적 방출 → 레이저 파쇄 불필요
3. 흰색(TiO2): 레이저 역산화 회피 → 면역경로 단독 제거
4. 세션 수 절반: 환자 부담 50% 감소
5. 흉터율 추가 감소: 레이저 에너지 의존도 하락
