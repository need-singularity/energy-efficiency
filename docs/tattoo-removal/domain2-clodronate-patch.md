# 도메인 2: 클로드로네이트 마이크로니들 패치

> 성숙도 40% | 3~5년 | 혁신성 90%

## 각 부품 성숙도

| 부품 | 성숙도 | 상태 |
|------|--------|------|
| 클로드로네이트 | 100% | 골다공증 치료, 특허 만료 |
| 리포솜 제형 | 100% | mRNA 백신으로 대중화, 시판 연구용 존재 |
| 용해성 마이크로니들 | 100% | 한국 세계 1위 (라파스, 쿼드메디슨) |
| **3부품 조합** | **40%** | **이 특정 조합 전임상 데이터 없음** |

## 왜 클로드로네이트가 "자연적 정밀 무기"인가

```
클로드로네이트 리포솜(200~400nm) 진피 주입
   ↓
"활발히 포식하는 세포"만 삼킨다
   ↓
잉크 보유 대식세포 = 활발히 포식한 대식세포
= 클로드로네이트 리포솜을 가장 잘 삼키는 세포
   ↓
잉크 들고 있는 놈이 독약도 가장 많이 먹는다
   ↓
선택적 사살 → 잉크 세포외 방출
```

비표적 세포 안전성:
- 섬유아세포: 리포솜 안 삼킴 → 무사
- 내피세포: 리포솜 안 삼킴 → 무사
- 호중구: 영향 없음
- 소극적 대식세포: 섭취율 낮아 상당수 생존

## 제품 설계

```
┌───────────────────┐
│ ┌─┬─┬─┬─┬─┬─┬─┐  │  접착 필름
│ │▼│▼│▼│▼│▼│▼│▼│  │  마이크로니들 어레이
│ └─┴─┴─┴─┴─┴─┴─┘  │  (600um, 11x11 = 121개)
│     2cm x 2cm     │  니들: 히알루론산 + CMC
└───────────────────┘
```

니들 1개 내부:
- 상위 300um: 클로드로네이트 리포솜 봉입층
- 하위 300um: 히알루론산 구조체

사용법:
1. 문신 부위에 패치 부착 (30초 엄지로 누름)
2. 15분 유지 → 니들 용해 → 클로드로네이트 리포솜 진피 전달
3. 패치 기저 제거
4. 24시간 후 레이저 시술

예상 제조 원가: 1~3만원/장
예상 판매가: 10~20만원/장 (시술 포함)

## 기술적 도전

1. **리포솜-니들 안정성**: 200~400nm 리포솜이 마이크로니들 건조 과정에서 안정성 유지?
   - 유사 사례: LNP in MN 연구 존재 → 동결건조 보호제(트레할로스) 사용
2. **진피 도달 후 분포**: 니들 121개 → 진피 내 균일 분포 가능?
3. **대식세포 사멸 정량**: 얼마나 빠르게, 얼마나 많이?
4. **레이저 병용 최적 간격**: 24시간 전? 48시간 전?
