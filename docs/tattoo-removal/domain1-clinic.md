# 도메인 1: 피코초+PFD R20 클리닉

> 성숙도 100% | 즉시 실행 가능 | 진입장벽: 장비+의사+공간

## 현실 상태

- 기술: 전부 시판 중, 조합만 하면 됨
- 한국 시장: R20+PFD 제대로 하는 곳 거의 없음
- 차별화: 세션당 4패스 → 기존 단일패스 대비 세션 수 40% 감소

## R20+PFD 시술 프로토콜 (1세션 70분)

| 시간 | 단계 |
|------|------|
| 0분 | 피부 사진 촬영 (더모스코프, 동일 조건) |
| 5분 | 국소 마취 (리도카인 크림 30~45분 또는 신경차단) |
| 50분 | 1차 레이저 패스 (색상별 파장 선택) |
| 51분 | 백색 서리 → PFD 패치 부착 |
| 53분 | 서리 소실 (2분) → 패치 제거 |
| 54분 | 2차 레이저 패스 |
| 55분 | PFD 재부착 |
| 57분 | 서리 소실 → 제거 |
| 58분 | 3차 레이저 패스 |
| 59분 | PFD |
| 61분 | 4차 레이저 패스 (마지막) |
| 65분 | 쿨링 + 항생제 연고 + 드레싱 |
| 70분 | 후처치 설명 + 귀가 |

## 색상별 파장 선택

| 파장 | 대상 색상 |
|------|----------|
| 1064nm (Nd:YAG) | 검정, 진한 파랑 |
| 532nm (KTP) | 빨강, 주황 |
| 755nm (Alexandrite) | 초록, 파랑 |
| 694nm (Ruby) | 초록, 청록 (기기 보유 시) |

## Fitzpatrick 타입별 프리셋 (6종 = n)

| 타입 | 파장 | 플루언스 | 스팟 | 주의 |
|------|------|---------|------|------|
| I | 전파장 | 표준 | 3~4mm | 없음 |
| II | 전파장 | 표준 | 3~4mm | 없음 |
| III | 1064 우선 | -10% | 3~4mm | 532nm 주의 |
| IV | 1064 단독 | -20% | 4~5mm | 저색소 모니터 |
| V | 1064 단독 | -30% | 5~6mm | 테스트패치 필수 |
| VI | 1064 단독 | -40~50% | 6mm | 간격 12주 |

한국 시장: Fitzpatrick III~IV 대다수 → 1064nm 중심
