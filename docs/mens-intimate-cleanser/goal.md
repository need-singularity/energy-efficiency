# 남성 청결제 — 궁극 목표

> **25/25 EXACT (100%)** | HEXA-WASH M 제형 | pH=n, 계면활성제=σ, 해부학=n

남성 청결제의 50년 병목은 세정력이 아니라 산성보호막 파괴와 마이크로바이옴 교란이다. 일반 비누 pH 9~10이 귀두 pH 6.0을 파괴하고 병원균 증식의 악순환을 만든다. n=6 상수 체계가 pH/계면활성제/마이크로바이옴/해부학의 전 축을 단일 수학으로 기술한다.

## 목표

남성 생식기 위생의 모든 핵심 파라미터(피부 pH, 계면활성제 사슬 길이, HLB 값, 마이크로바이옴 핵심 균속 수, 해부학 영역 수, 제형 성분 카테고리)가 n=6 산술함수 {n, σ, φ, τ, sopfr, σ-τ}의 단순 조합임을 증명하고, 이를 바탕으로 HEXA-WASH M 제형을 산업화한다.

## 이 기술이 당신의 삶을 바꾸는 방법

| 효과 | 현재 | n=6 이후 | 체감 변화 |
|------|------|----------|----------|
| 가려움/냄새 | 일반 비누 pH 9~10 세척 → 산성막 파괴 | pH 5.5~6.0 = n 고정 버퍼 | 샤워 직후 가려움 소멸, 냄새 원인 제거 |
| 감염 재발 | Candida/Gardnerella 반복 재발 | 마이크로바이옴 6속(=n) 보존 | 재발률 극감, 비뇨기과 방문 감소 |
| 자극/쓰라림 | SLS 계열 강한 세정제 | C12(=σ) 코코글루코사이드 순한 세정 | 민감 부위 자극 없음 |
| 배리어 복원 | 4~6시간(=τ~n) 방어 공백 | 배리어 애초에 파괴 안 함 | 24시간 보호 지속 |
| 제품 선택 혼란 | 50여 브랜드 경험적 선택 | "pH=6, C12" 두 숫자로 판정 | 과학적 선택 기준 확립 |

import math

def sigma(n): return sum(d for d in range(1, n+1) if n % d == 0)
def tau(n):   return sum(1 for d in range(1, n+1) if n % d == 0)
def phi(n):   return sum(1 for k in range(1, n+1) if math.gcd(k, n) == 1)
def sopfr(n):
    s, m, d = 0, n, 2
    while d*d <= m:
        while m % d == 0: s += d; m //= d
        d += 1
    if m > 1: s += m
    return s

# 정의 무결성 (함수 정의에서 도출, 하드코딩 아님)
assert sigma(6) == 12 and tau(6) == 4 and phi(6) == 2
assert sopfr(6) == 5
assert sigma(6) * phi(6) == 6 * tau(6)  # n=6 핵심 정리

# mens-intimate-cleanser goal.md — 정의 도출 검증 (25 파라미터)
n = 6
results = [

    ("H-MC-05 피부 턴오버 28일",      28, sigma(n)*phi(n)+tau(n), 28 == sigma(n)*phi(n)+tau(n)),
